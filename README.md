# SHOTSMITH Labs

Luxury biotech wellness single-page experience built with React + Vite.

## Stack

- React + TypeScript
- Lenis smooth scrolling
- GSAP ScrollTrigger for cinematic scroll scenes
- Framer Motion for section reveals

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Visuals use CSS placeholders for bottle and liquid simulations. Replace with real renders when available.
- Fonts are referenced by family name only. Add actual font files or self-hosted font setup for production.
- The experience includes 10 sections mapped to the provided SHOTSMITH Labs architecture.
